# Лабораторная работа №1

## По дисциплине «Исследование операций»

---

## 📌 Тема

**Исследование множества опорных планов системы ограничений задачи линейного программирования (ЗЛП) в канонической форме**

---

## 🎯 Цель работы

Изучить метод Гаусса–Жордана и операцию замещения, а также освоить их применение для:

* отыскания множества допустимых базисных видов системы линейных уравнений;
* решения задачи линейного программирования методом прямого перебора опорных решений.

---

## 📋 Задания

### 🔹 Задание 1

Составить программу для отыскания всех базисных видов системы линейных уравнений.

✔ Выполнено:

* Реализован алгоритм поиска базисных решений на Python
* Построены **3 блок-схемы**
* Получены результаты работы программы

---

### 🔹 Задание 2

Организовать отбор опорных планов среди всех базисных решений, а также нахождение оптимального опорного плана методом прямого перебора.

✔ Выполнено:

* Реализован алгоритм отбора допустимых решений
* Реализован перебор опорных планов
* Найден оптимальный план
* Построены **6 блок-схем**

---

### 🔹 Задание 3

Решить задачу вручную (вариант №8).

#### Исходная система:

```
6x1 - 2x2 + 2x3 + 5x5 = 2
-x1 - 13x2 + 7x3 + 11x4 = 17
4x1 - 7x2 + 2x3 + x4 + 3x5 = -6
5x1 - 15x2 + 9x3 + 11x4 + 5x5 = 19
```

---

## 🧩 Используемые методы

* Метод Гаусса–Жордана
* Операция замещения
* Перебор базисов
* Проверка допустимости решений
* Поиск оптимального решения
